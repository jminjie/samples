You can use your own samples if you statically host them. Just include a config.json file which sets the mapping. e.g.

        {
        "A1": "A1.mp3",
        "A2": "A2.mp3",
        "A3": "A3.mp3"
        }

The mapped samples are used to extrapolate to other pitches.
